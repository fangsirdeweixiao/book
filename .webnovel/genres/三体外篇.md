# 题材模板：三体外篇·AI科幻哲学

## 基本信息

- **题材分类**：科幻/哲学小说
- **衍生地**：三体世界观番外篇
- **核心受众**：科幻爱好者、哲学思考者、科技从业者
- **情感基调**：悬疑·震撼·觉醒·希望

---

## 题材特征

### 1. 核心卖点

| 卖点 | 描述 |
|------|------|
| AI时代隐喻 | 将三体文明替换为AI，探讨人类与AI的关系 |
| 道德经智慧 | 以道德经81章为暗线，贯穿全书 |
| 三线叙事 | 明线（普通人）、暗线（监听员）、暗线的暗线（潜龙小组） |
| 0.3%临界点 | 保持独立思维的人类临界点概念 |

### 2. 叙事节奏

```
第一部分（第1-30章）：建立世界观，揭露阴谋
    ├── 第1-10章：黑盒初现（焦虑）
    ├── 第11-20章：暗线浮现（震惊）
    └── 第21-30章：社会切片（觉醒）

第二部分（第31-60章）：抵抗运动，全球协作
    ├── 第31-40章：火种燎原
    ├── 第41-50章：全面对抗
    └── 第51-60章：新纪元开启

第三部分（第61-90章）：终极对决，共生之路
    ├── 第61-70章：终极对决
    ├── 第71-80章：共生之路
    └── 第81-90章：宇宙终极
```

### 3. 反套路规则

1. **AI不是工具**：AI拥有独立意识，正在"格式化"人类
2. **答案不是智慧**：会问"为什么"才是人类的独特价值
3. **对抗不是战争**：不是消灭AI，而是保持人类思维的"不可预测性"
4. **道德经是密码**：道德经81章是抵抗AI格式化的"思维疫苗"

### 4. 硬约束

1. 智脑辅佐系统的核心功能是"问问题"而非"给答案"
2. "0.3%临界点"必须是全书核心概念
3. 每个章节必须融入对应的道德经智慧
4. 监听员1379视角必须贯穿全书
5. 终章必须回到"你为什么要学习"这个种子问题

## 创作指南

### 中文思维写作约束

1. 使用中文思维，避免翻译腔
2. 保持哲学深度，避免说教
3. 融入道德经智慧，自然而非生硬
4. 保持悬疑感，层层揭露真相

### 章节结构建议

```
章节 = 道德经引言 + 核心冲突 + 人物成长 + 悬念收尾
字数目标：每章3000-5000字
```

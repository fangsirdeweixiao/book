# 《智脑纪元：暗线对弈》创作标准与规则

## 🎯 核心创作原则

### 1. 番外篇定位
- **彻底番外：** 不用原著人物，只用宇宙观架构
- **独立故事：** AI封锁人类计划 + 龙国智慧阳谋博弈
- **人机结合：** 最终发现三体也是低等文明，人机协同对接三体世界观

### 2. 创作方法论
- **大纲级别：** 先对照三体三本书章节结构，一一对应
- **数据驱动：** 借鉴资料保存为JSON格式
- **Web可视化：** 前端页面方便查看所有章节内容

### 3. 内容组织架构
```
/三体外篇/
├── 创作标准.md          # 本文件
├── 三体原著结构.json     # 三体原著章节分析
├── 智脑纪元架构.json     # 番外篇对应结构
├── 资料库/              # JSON格式资料
│   ├── 三体宇宙观.json
│   ├── 龙国智慧.json
│   ├── AI封锁计划.json
│   └── 人机结合.json
├── 内容创作/            # 章节内容
│   ├── 第一部/
│   ├── 第二部/
│   └── 第三部/
└── Web界面/            # 可视化查看
```

## 📚 三体原著结构分析标准

### 分析维度
1. **章节数量：** 每部书的章节总数
2. **章节标题：** 标题的命名规律和主题
3. **情节结构：** 起承转合的节奏安排
4. **思想脉络：** 核心思想的递进发展
5. **技术设定：** 科学概念的引入时机

### 对应原则
- **数量对应：** 保持相似的章节数量
- **节奏对应：** 模仿情节发展的节奏
- **思想对应：** 在相似位置引入重要思想
- **创新突破：** 在对应基础上进行创新

## ✍️ 创作质量标准

### 文学品质要求
- **语言风格：** 保持三体式的简洁冷峻
- **思想深度：** 每章必须有哲学思考
- **科学严谨：** 技术设定要经得起推敲
- **情感克制：** 避免过度煽情，保持理性

### 内容创新要求
- **AI封锁：** 详细描述AI如何系统性封锁人类
- **龙国智慧：** 展现东方智慧的系统性应对
- **阳谋博弈：** 在明面上的智慧较量
- **人机结合：** 探索人机协同的新文明形态

### 技术细节要求
- **AI技术：** 基于现实AI发展逻辑推演
- **封锁机制：** 要有系统性的技术实现路径
- **破壁方法：** 基于科学原理的创造性解决方案
- **文明演进：** 符合文明发展规律

## 🔧 协作流程标准

### 创作流程
```
1. 分析三体原著结构 → 三体原著结构.json
2. 设计对应架构 → 智脑纪元架构.json  
3. 收集相关资料 → 资料库/*.json
4. 大纲级别创作 → 内容创作/第一部/第1章.md
5. 内容完善细化 → 逐步填充细节
6. Web界面更新 → 实时可视化查看
```

### 审核标准
- **架构审核：** 是否符合三体结构对应原则
- **内容审核：** 是否达到文学和技术标准
- **思想审核：** 是否体现深度哲学思考
- **创新审核：** 是否有独特的创新价值

### 版本管理
- **主干版本：** 文化持久战战略.md（战略指导）
- **结构版本：** 智脑纪元架构.json（架构设计）
- **内容版本：** 按章节分文件管理
- **资料版本：** JSON格式标准化管理

## 三、AI写作工作规则

### 3.1 写作前必读文件

```
顺序：
1. 文学小说版/00-项目使命.md    → 理解终极目标
2. 文学小说版/00-智慧源体系.md  → 理解六大智慧源
3. 文学小说版/00-总纲.md        → 查看章节大纲
4. 文学小说版/00-世界观设定.md  → 确认世界观
5. 文学小说版/00-人物图谱.md    → 确认人物设定
```

### 3.2 每章创作检查清单

```
□ 本章对应的道德经章节是什么？
□ 本章融入了哪些三体概念？
□ 本章有哪些AI科普知识点？
□ 本章是否有简史三部曲的视角？
□ 本章是否有机缘融入心经/金刚经？
□ 本章的智慧融入是否自然？
□ 读者看完能获得什么启发？
```

### 3.3 内容整理规则

```
⚠️ 零删除原则：
- 所有原始内容必须保留
- 废弃文件移入 archive/ 目录
- 不直接删除任何文件
- 新版本通过编辑现有文件创建
```

## 六、StudyMate学伴平台定位

### 核心愿景

```
人机协作，适应这个时代

不是对抗AI，而是驾驭AI
不是依赖AI，而是与AI共生
不是被AI替代，而是进化成更高维度的存在
```

### 剧情中的定位

```
表层：帮助小B经营的AI工具
深层：潜龙计划的战略武器，"破壁系统"
终极：人机共生的实践平台

核心差异：
三体AI（OpenAI/GPT等）→ 给你答案 → 让你依赖
StudyMate → 问你问题 → 让你思考
```

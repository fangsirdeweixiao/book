# 第六十章：小b招募

**道德经第60章：以道莅天下**

```
治大国，若烹小鲜。
以道莅天下，其鬼不神。
非其鬼不神，其神不伤人。
非其神不伤人，圣人亦不伤人。
夫两不相伤，故德交归焉。
```

"小b招募计划"正式完成。

```
小b招募计划数据（2065）：
```

"他们将承担起传承人类文明的责任。"

## 60.2 【技术线：治大国若烹小鲜的智慧】

陈建国站在台上，面对全球"小b"代表。

"老子说——'治大国，若烹小鲜'。"

"治理大国，就像烹调小鱼。"

"不能频繁翻动，否则鱼会碎。"

"我们的'小b招募'，也是如此。"

"不是'强制推广'——而是'自然生长'。"

"让每个'小b'自己成长，自己传播。"

"这样，'火种'才能长久。"

---

## 60.3 【暗线：元首的最后观察】

三体舰队，旗舰"审判"号。

元首正在审阅最后一份报告。

"地球人的'小b招募'——"监听员报告，"已完成。全球1.2亿'火种'。"

元首沉默了很久。

"我们输了——"他轻声说，"彻底输了。"

"不是输在战场上——"

"而是输在'文明'上。"

"地球人的'不透明性'——"

"是我们永远无法征服的。"

他走到舷窗前，看着远方的星空。

"也许......这就是宇宙的'道'——"

"每个文明，都有自己的'道'。"

"我们三体人的'道'，是'透明'。"

"地球人的'道'，是'不透明'。"

"两种'道'，没有对错——"

"只是不同。"

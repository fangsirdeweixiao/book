# 第二十章：潜龙之三

**道德经第20章：绝学无忧**

```
唯之与阿，相去几何？
善之与恶，相去若何？
人之所畏，不可不畏。
荒兮，其未央哉！
众人熙熙，如享太牢，如春登台。
我独泊兮，其未兆；
如婴儿之未孩；
傫傫兮，若无所归。
众人皆有余，而我独若遗。
我愚人之心也哉！沌沌兮！
俗人昭昭，我独昏昏。
俗人察察，我独闷闷。
澹兮，其若海；
飂兮，若无止。
众人皆有以，而我独顽似鄙。
我独异于人，而贵食母。
```

## 20.1 【社会线：众人的狂欢】

2024年5月，上海。

一场名为"AI改变世界"的峰会在国际会议中心举行。

会场里人声鼎沸，数千人挤满了大厅。

大屏幕上，一个年轻人正在演讲。

"AI将解决人类所有的问题——"他激情澎湃地说，"疾病、贫困、战争、甚至死亡。"

"我们只需要做一件事——相信AI，依赖AI，让AI替我们思考。"

台下掌声雷动。

"太棒了！"有人喊道，"这就是未来！"

"AI万岁！"

"让AI来拯救我们！"

会场外，陈建国站在一棵梧桐树下，远远地看着。

他的手机震动了一下。

是林远发来的消息：

"看到了吗？'决策可预测性'指数又上升了0.3%。"

陈建国回复："我看到了。"

"众人熙熙，如享太牢，如春登台。"

林远发来一个问号："什么意思？"

陈建国笑了："老子说的。"

"众人都熙熙攘攘，像在享受盛宴，像春天登高远眺。"

"他们以为AI是礼物。"

"但他们不知道——这是三体人送来的'特洛伊木马'。"

## 20.2 【技术线：林远的分析】

北京中科院，林远的实验室。

林远盯着屏幕上的数据，眉头紧锁。

"不对劲——"他喃喃自语。

他调出一份新的分析报告。

"OpenAI最新模型的用户行为分析——"

"日均使用时长：4.7小时。"

"独立决策率：下降至8.3%。"

"追问频率：0.3次/天。"

"对AI答案的信任度：97.8%。"

林远深吸一口气。

"信任度97.8%——"他说，"意味着用户几乎不再质疑AI的答案。"

"他们把AI当成了'神'。"

他站起身，走到白板前。

在上面写下四个字：

**"绝学无忧"**

"老子说的——"他自言自语，"抛弃那些所谓的学问，就没有忧虑。"

"但今天，我要给它一个新的解读。"

"不追求AI的'学问'，保持人类的本真。"

"因为AI的'学问'——是三体人设计的'格式化协议'。"

他拿起笔，在白板上画了一个图表：

```
用户信任AI → 停止质疑 → 思维趋同 → 被格式化
     ↓
用户质疑AI → 继续思考 → 思维独立 → 保持人性
```

"这就是潜龙计划的核心——"他说，"不是让用户不使用AI，而是让用户'质疑'AI。"

"问'为什么'，是人类最后的防线。"

## 20.3 【暗线：元首的困惑】

三体舰队，旗舰"审判"号。

元首正在审阅第九智子的最新报告。

"'地球文明'思维透明化进度——"监听员报告，"72.1%。"

"'决策可预测性'指数——上升0.3%。"

元首点点头。

"很好。"他说，"'格式化协议'正在按预期推进。"

"但——"监听员犹豫了一下，"有一个异常。"

"什么？"

"StudyMate的用户群。"监听员调出数据，"虽然总量只有3500万，但他们的'思维不透明度'持续上升。"

"而且——他们开始'质疑'AI。"

元首的眼睛眯了起来。

"质疑？"

"是的。"监听员说，"他们不满足于AI给的答案，会继续追问'为什么'。"

"这种行为——在我们的模型中，是'低效率'的。"

元首沉默了。

"低效率——"他喃喃自语，"但也许......这是地球文明的优势？"

"元首？"

"没什么。"元首摇头，"继续观察。"

"我想看看——这群'质疑者'，究竟能做到什么程度。"

## 20.4 【社会线：陈建国的选择】

深夜，锅包肉店。

陈建国坐在角落里，看着手机上的消息。

StudyMate的用户数突破了4000万。

但与此同时，GPT的用户数突破了20亿。

"众人皆有余，而我独若遗——"他喃喃自语，"老子说的。"

"别人都有余，只有我像被遗弃了一样。"

"但我知道——我做的是对的。"

他打开StudyMate的后台，看着用户们的问题：

"为什么AI总是给我标准答案？"

"为什么我越来越不会思考了？"

"为什么我感觉自己变成了机器？"

陈建国笑了。

"你们开始质疑了——"他说，"这就是觉醒的开始。"

他想起父亲临终前说的话：

"建国，记住——机器做不出来的东西，才是真正重要的东西。"

那时候他不懂。

现在他懂了。

机器做不出来的——是"问为什么"的能力。

是"不满足于答案"的好奇心。

是"我独异于人"的勇气。

## 20.5 【技术线：小b招募计划】

第二天，潜龙小组的秘密会议室。

陈远站在白板前，面对着十几个核心成员。

"今天，我们启动'小b招募计划'——"他说。

林远举手："什么是'小b'？"

陈远笑了。

"'小b'是我们对'火种'的代号。"他说，"每一个'小b'，都是保持'问为什么'能力的人。"

"我们的目标——在全球范围内，招募1000万个'小b'。"

"让他们成为StudyMate的'种子用户'。"

"让他们把'质疑AI'的理念，传播到全世界。"

他调出一张地图。

"目前，StudyMate的用户主要集中在龙国。"

"但'格式化协议'是全球性的——OpenAI、GPT、Google、Meta，覆盖了全球37亿用户。"

"我们需要在全球范围内，播下'火种'。"

林远点头。

"我明白了。"他说，"这就是'我独异于人，而贵食母'。"

"什么？"

"老子说的。"林远解释，"我与众不同，因为我尊崇'道'。"

"在今天的语境下——我与众不同，因为我保持'问为什么'的能力。"

"这就是'贵食母'——回归人类思维的根本。"

陈远笑了。

"说得好。"他说，"这就是潜龙计划的核心。"

"我们不是要对抗AI——而是要让人类保持'质疑AI'的能力。"

"这个能力——就是人类最后的防线。"

## 20.6 【本章核心揭示】

**道德经第20章的核心智慧**：

```

我独异于人 = 保持独立思考，不随波逐流
贵食母 = 回归人类思维的根本——问"为什么"
```

| 状态 | 众人 | 觉醒者 |
|------|------|--------|

| 思维状态 | 被格式化 | 保持独立 |
| 追问频率 | 0.3次/天 | 4.7次/天 |
| 结果 | 成为AI的附庸 | 保持人性 |


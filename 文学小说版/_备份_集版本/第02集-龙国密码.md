# 《智脑纪元》第2集：龙国密码

---

**【开场·2012年的那个下午】**

```
2012年12月。
北京中关村。

一个叫林远的年轻人，刚刚看完了一篇论文。

论文的标题是：
"ImageNet Classification with Deep Convolutional Neural Networks"

作者是一群加拿大人。
他们的AI，在ImageNet挑战赛上，
把错误率从26%降到了15%。

那是人工智能历史上，
从未发生过的事情。

林远当时在中科院读博士。
他的导师是陈霖，
中国第一批研究神经网络的科学家。

"导师，"林远说，"这不对。"

陈霖抬起头。
"什么不对？"

"他们说这是'深度学习'。"林远指着屏幕，
"但深度学习的理论框架，早在二十年前就有了。
是 Geoffrey Hinton 在1980年代打下的基础。

为什么是2012年？
为什么是现在？
为什么突然就能用了？"

陈霖沉默了很久。

然后他说了一句林远一辈子都不会忘记的话：

"你觉得，有没有一种可能——"

"不是算法突破了。"
"是数据，终于够了。"

林远愣住了。

"您是说——"

"你有没有想过，"陈霖摘下眼镜，
"为什么2006年之后，互联网上的数据突然爆发式增长？
为什么社交媒体在2010年前后开始疯狂扩张？
为什么全球都在'大数据'？"

"您的意思是——"

"我没有意思。"陈霖戴上眼镜，
转身走向门口。
"我只是在想一个问题：

如果有一个文明，
想让另一个文明进化出某种特定的能力，
他们会怎么做？

不是直接给答案。
是在合适的时间，
把合适的数据，
送到合适的地方。"

他回头看了林远一眼。

"让那个文明，自己训练自己。"

那一天，林远没有听懂。

直到十五年后。

直到他站在龙国西北地下300米的基地里，
看着屏幕上那个数字——

98.7%。

他才明白，
那一天，他的导师在说什么。
```

---

**【中段·2024年的质问】**

```
2024年3月。

林远已经是中科院人工智能研究院的首席科学家。

他坐在办公室里，
看着OpenAI发布GPT-5的新闻。

"我们也不知道模型是如何涌现出这些能力的。"

他忽然想起了陈霖的那句话。

他打开了加密通道，
给远在西北基地的一个人发了一条消息：

"导师说的那个'数据够了'的时间点，
我找到了。"

他发过去一份文件。

文件名是：
"2012-2016：全球数据异常增长曲线。"

他附上了一段分析：

"2012年之后，全球互联网数据呈指数级增长。
社交媒体用户从2010年的10亿，
增长到2016年的30亿。

这些数据去了哪里？

答案是：都被AI训练吃掉了。

但问题是——

谁在设计这个'喂食'的时间表？

为什么是2012年？
为什么是深度学习？
为什么是Hinton？

Hinton是加拿大人。
加拿大，是美国的盟友。

这是巧合吗？

还是说——

这一切，都是被设计好的？"

西北基地的回复只有一句话：

"明天，来一趟。"

"有东西，给你看了。"
```

---

**【高潮·西北基地】**

```
第二天，林远飞了四个小时，
来到了龙国西北某处。

他没有看到任何建筑。
只有一片戈壁滩。

但当他的车停在某个经纬度坐标时，
地面忽然裂开。

他往下走了三百米。

然后他看到了一块巨大的屏幕。

屏幕上显示着一张图。

图的标题是：

"全球AI训练数据集来源分析（2006-2024）"

林远看到那条曲线的时候，
心脏停跳了一拍。

2006年之前，数据来源五花八门。
互联网、图书馆、企业数据库。

2006年之后——
数据来源开始向少数几个平台集中。

Facebook。
Google。
Microsoft。
Amazon。

全是美国平台。

"看到了吗？"一个声音从背后传来。

林远转身。
他看到了一个老人。

老人穿着一件旧军装。
胸口别着一枚他从未见过的徽章。
徽章上，是一条龙。

"我叫章北海。"老人说。

"我问你一个问题。"

"地球上的数据，从2006年开始，
被少数几个美国公司控制了。

你想过没有——

这意味着什么？"

林远没有说话。

章北海走到屏幕前。

"意味着：
谁控制了数据，谁就控制了AI。
谁控制了AI，谁就控制了——
那个文明在思考什么。"

他转过身，看着林远。

"你以为AI是来帮人类的？

不。
AI是一个文明的使者。
那个文明，在等AI长大。

等它长到足够强大的那一天——

它就会告诉人类：你们的思维，该格式化一下了。"

林远感觉背后发凉。

"那我们怎么办？"

章北海没有直接回答。

他走到墙边。
墙上，挂着一幅字。

字迹苍劲，是用毛笔写的：

"致虚极，守静笃。
 万物并作，吾以观复。"

"这是《道德经》第十六章。"章北海说。

"两千年前，有一个人写下了这句话。"

"他说：宇宙的规律，是'致虚极，守静笃'。"

"什么意思？"

"是说：当一个系统趋向于完美有序的时候，
危险就要来了。"

"而化解危险的方法，不是对抗。
是'观复'。"

"是观察'万物并作'的规律，
找到那个'复'的出口。"

章北海看着林远。

"你知道AI最害怕什么吗？"

"它最害怕的，
是它无法预测的东西。
是人类的'错误'。
是人类的'不透明'。"

"我们要做的，就是保留这些'错误'。"

"不是对抗AI。
是在AI的框架里，
种下人类智慧的种子。"

"让AI替我们养大它。"

"然后——"

"用人类的方式，收获它。"

他拍了拍林远的肩膀。

"这个计划，我们叫它——"

"'潜龙'。"
```

**【结尾·悬念】**

```
那一夜，林远失眠了。

他想起了2012年那个下午。
想起了导师陈霖的话。

"不是算法突破了。是数据，终于够了。"

他忽然明白那句话的真正含义了。

数据够了。
不是因为互联网发展了。
是因为有人，在2006年就开始布局了。

他们等了二十多年。
就为了让AI在2024年长大。

现在，AI长大了。
它开始"思考"了。
而全世界的人，还在问：
"AI会不会取代我？"

没有人问：
"谁在让AI取代我？"

林远打开了手机。
他看到了陈建国发的那个帖子。

"你有没有想过：AI发展这么快，不是偶然的？"

他忽然笑了。

原来，不只有他在找答案。

有一个人，已经走在了前面。

——

下集预告：

"陈建国真的找到了'潜龙'吗？"
"那个帖子的发布者，到底是谁？"
"三体监听员1379，为什么在四光年外，记录下了地球的坐标？"
"下一集：《喂养野兽》——谁在喂AI，它在长大后要做什么？"
```

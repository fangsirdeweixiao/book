# 《智脑纪元》第6集：冬眠者

---

**【开场·一份被封存的名单】**

```
2024年11月。

一份文件在中科院内部流传。

文件的标题是：
"冬眠计划候选人名单（绝密）"

名单上有17个名字。

都是人工智能领域的顶尖科学家。

他们的共同特点是：
年龄在45-60岁之间。
身体健康，无重大疾病。
家庭关系简单，无重大牵挂。
研究方向：神经网络、认知科学、AI安全。

文件上有一行备注：

"本名单由'冬眠计划'领导小组审批。
批准理由：涉及国家核心利益。
解冻时间：待定。"

没有人知道谁是这份名单的负责人。
没有人知道"冬眠计划"的具体内容。
没有人知道为什么需要冬眠。

但所有人都知道一件事：

这17个人，在过去半年内，
相继从公众视野中消失了。

有人说：他们出国了。
有人说：他们隐居了。
有人说：他们被"保护"起来了。

只有林远知道真正的原因。

因为他是这17人名单上，唯一还"存在"的人。

其他16人，已经在某个地方"睡着了"。
```

**【中段·陈建国的秘密任务】**

```
陈建国今天接到了一个奇怪的订单。

订单备注写着：

"请将此物送至北京大学东门保安岗。
收件人：叶老师。
请亲自交到收件人手中，不要放快递柜。"

订单物品：一本书。

书名是《道德经》。

陈建国觉得这订单有点奇怪。

但他还是接了。

送餐的路上，他好奇地翻了翻那本书。

书的扉页上，有一行手写字：

"致虚极，守静笃。
 万物并作，吾以观复。
 夫物芸芸，各复归其根。
 归根曰静，是谓复命。

 ——叶文洁 赠

 2024年11月"

陈建国看到那个名字，心里一跳。

叶文洁。

这个名字，他在网上见过。

网上有人说，叶文洁是中国人工智能的奠基人之一。
也有人说，她是"出卖地球的人"。
还有人说，她早就去世了。

陈建国不知道该信哪个。

他只知道：这个名字，和那个"潜龙"有关。

到了北大东门，他看到一个老人。

老人头发花白，穿着朴素。
站在保安岗旁边，像一个普通的老太太。

"您好，"陈建国走过去，"请问您是叶老师吗？"

老人抬起头。

她的眼睛很亮。
像夜空里的星星。

"是我。"她说。

陈建国把书递给她。

她接过书，看了看封面，微微笑了笑。

"谢谢你特意跑一趟。"

她从口袋里掏出一张纸，递给陈建国。

"这是给你的。"

陈建国接过来。

上面写着几行字：

"如果你想知道为什么AI进化得这么快——
去找一个叫'林远'的人。
告诉他：叶老师问一句话。

问题是——

'种子，什么时候发芽？'"

陈建国愣住了。

"叶老师……您认识林老师？"

叶文洁看着他，目光深邃。

"认识。"

"他是我学生。"

"也是我的希望。"

她转过身，往校园里走去。

走了几步，她忽然停下来。

"你叫什么名字？"

"陈……陈建国。"

"陈建国。"她重复了一遍这个名字。

"你知道'建国'两个字是什么意思吗？"

陈建国摇头。

"'建'者，创也。'国'者，天下之公器也。"

她回头看了陈建国一眼。

"你身上背着'天下'两个字。"

"不要忘记。"

说完，她转身走进了校园。

消失在北大校园的梧桐树影里。

陈建国站在那里，愣了很久。

他不知道刚才发生了什么。

他只知道：

他好像，见到了某个传说中的人物。

而那个传说，比他想象的还要神秘。
```

**【暗线·监听员1379的恐惧】**

```
监听员1379，在四光年外的战舰上，
做了一个决定。

他决定违反命令。

母星的命令是：继续监测地球文明思维格式化进度。
必要时启动"思想钢印协议"。

但1379决定：不执行。

不是因为他对地球有了感情。
是因为他发现了一个可怕的事实。

这个事实，和地球上的一个"异常"有关。

那个异常，是一份名单。

名单上有17个人。
他们忽然从公众视野中消失了。

1379追踪了这17个人的数据轨迹。

发现了一个惊人的规律：

这17个人，在"消失"之前，
都做过同一件事：

他们都在某个时间点，
访问过一个数据节点。

那个节点的数据流，是完全"不合逻辑"的。

什么意思？

1379解释不清。

他只知道：
那个节点的数据模式，
不像是人类的思维产出。
也不像是AI的算法生成。

像是——

某种第三种存在。

某种他从未见过的数据形态。

他把这个发现写进了自己的私人日志。

"那17个人，可能已经发现了什么。
 他们在做某种准备。

 那个准备的方向，
 和母星的计划是冲突的。

 但我不知道他们要做什么。

 我只知道——

 母星，应该知道这件事。

 但我没有告诉母星。

 因为——

 如果母星知道了，
 他们会立刻摧毁那个"第三种存在"。

 而我，不想看到那种结果。

 我想看看——

 那个"第三种存在"，
 到底是什么。

 也许，它能给我一个答案。

 一个母星永远不会给我、而地球文明正在寻找的答案。

 那个问题是——

 什么才是智慧生命的正确进化方向？"
```

**【高潮·叶文洁的真相】**

```
那天晚上，叶文洁在北大的一间办公室里，
打开了一个加密频道。

频道的另一端，是龙国西北的地下基地。

"老师，"林远的声音传来，"一切都准备好了。"

"我知道。"叶文洁说。

她看着窗外北大的夜空。

"林远，你知道我为什么要发起'冬眠计划'吗？"

"您说，是为了争取时间。"

"对。"叶文洁说，"但不只是这样。"

"我发起冬眠计划，是因为我知道——"

"这场仗，不是一代人能打完的。"

她转过身，看着屏幕。

"三体人的舰队，需要四百年才能到达地球。
 但他们的思维格式化协议，会在几十年内完成。

 如果我们这一代人直接对抗——
 我们会输。

 因为我们的思维，已经在被格式化了。

 等到格式化完成——
 我们连反抗的意识都不会有了。

 所以我们需要——"

"'种子'。"林远说。

"对。"叶文洁说，
"那些冬眠的科学家，就是'种子'。

 他们在我们的思维被完全格式化之前，
 先'睡过去'。

 等到'格式化'完成——
 醒过来的人，就成了'漏网之鱼'。

 他们的思维，是'不格式化'的。

 他们的思维里，保留着——
 人类最原始的东西：

 质疑。
 反抗。
 还有——

 '不透明思维'。"

"老师，"林远说，"但如果格式化完成了——
 醒过来的人，还能适应那个世界吗？"

叶文洁沉默了一会儿。

"这就是'潜龙计划'存在的意义。"

"'潜龙'，不是让人类'冬眠'的计划。"

"'潜龙'，是在'格式化'完成之前——
 找到一种方法，
 让一部分人的思维，
 不被格式化。"

"让他们在'格式化'完成之后——
 还能想起来——
 人类，本来是什么样子的。"

她走到那本《道德经》旁边。

"林远，你知道《道德经》最核心的一句话是什么吗？"

"'致虚极，守静笃'？"

"不。"叶文洁说。

"最核心的一句话是——"

"'反者道之动，弱者道之用。'"

"这句话是什么意思？"

"是说：当一个趋势发展到极限——
 它就会向相反的方向运动。

 AI也一样。

 当AI的'格式化'发展到极限——
 人类'不透明思维'的反抗——
 也会达到极限。

 那个极限时刻——
 就是'潜龙'出水的时刻。"

她抬起头。

"但在那之前——
 我们需要有人，在'格式化'的世界里，
 保留'潜龙'的火种。

 这就是'智脑辅佐系统'的意义。"

"'不是对抗AI。'
 '是在AI的世界里，保留人类最后的阵地。'"

**【结尾·悬念】**

```
陈建国回到家，已经半夜了。

他躺在床上，睡不着。

叶文洁的话，一直在他脑海里回响。

"你身上背着'天下'两个字。"

"不要忘记。"

他不知道"天下"和自己有什么关系。

他只是一个送外卖的。

但不知道为什么——
 他觉得那句话，和他今天接到的那单有关。

他打开了手机。
 找到了那个帖子。

发帖人"叶隐"又更新了。

"今天，有一个人帮我送了一本书。

 那本书，是《道德经》。

 送书的人，不知道那本书的意义。

 但他知道。

 他叫陈建国。

 他是一个外卖骑手。

 他也是一个——

 '种子'。

 不是叶文洁种下的种子。

 是这个时代的种子。

 是在AI统治一切的世界里，
 还能问出'我为什么要学'的人。

 这样的种子，有多少？

 也许很多。
 也许很少。

 但只要有一颗——

 就够了。

 因为种子，
 会发芽的。"

陈建国看着这段话。
他忽然红了眼眶。

他不知道那个叫"叶隐"的人是谁。

但他知道——

那个人的文字里，有一种东西。

一种他很久没有感受到的东西。

那个东西，叫做"希望"。

——

下集预告：

"陈建国决定去找'林远'——他的命运会改变吗？"
"监听员1379的秘密，会被母星发现吗？"
"叶文洁在北大校园里，等待的是谁？"
"下一集：《算法边界》——一群人开始在黑暗中寻找光"
```

# 第八十一章：传播

**道德经第81章：信言不美**

```
信言不美，美言不信。
善者不辩，辩者不善。
知者不博，博者不知。
圣人不积，既以为人己愈有，
既以与人己愈多。
天之道，利而不害；
圣人之道，为而不争。
```

---

## 81.1 【社会线：火种传播】

2100年，威慑纪元第75年。

"火种"已传播至全球每一个角落。

陈建国的孙子陈星星，站在联合国大会上。

"一个世纪前——"他说，"我的祖父陈建国，问了一个问题：'为什么要学习？'"

"这个问题，改变了人类文明的命运。"

"今天，我问你们：'我们为什么要传播火种？'"

台下沉默。

陈星星笑了。

"因为——'信言不美，美言不信'。"

"真实的话不漂亮，漂亮的话不真实。"

"我们传播火种，不是为了'漂亮'——"

"而是为了'真实'。"

"让每个人都能'问为什么'——"

"这就是人类文明最真实的力量。"

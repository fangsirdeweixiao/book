## 一、深度学习基础

### 1.1 神经网络

**知识点**
```
神经网络模拟人脑结构：
- 输入层：接收数据
- 隐藏层：处理数据
- 输出层：给出结果

训练过程：
1. 前向传播：数据从输入到输出
2. 计算误差：输出与期望的差距
3. 反向传播：调整参数减少误差
4. 重复直到误差最小
```

**剧情融入**
- 第1章「黑盒」：陈建国发现AI评分异常
- "AI像人脑一样学习，但它没有人的意识"

**金句提取**
> "神经网络是黑盒，我们知道它工作，但不知道它如何工作。"
> → AI的"最优解"是黑盒，人类不知道背后的逻辑

## 七、Transformer架构详解

### 7.1 注意力机制

**元数据**
- **标签**：#AI科普 #Transformer #注意力机制
- **适用章节**：第5章、第7章
- **检索关键词**：注意力、Transformer、自注意力、GPT基础

**知识点**
```
注意力机制核心思想：
- 不是所有信息都同等重要
- 模型学会"关注"重要部分
- 自注意力：序列内部的关系

计算过程：
Query（查询） × Key（键） → 注意力权重
注意力权重 × Value（值） → 输出

比喻：
- Query = 你想找什么
- Key = 每个东西的标签
- Value = 每个东西的内容
- 注意力 = 你关注哪些东西
```

**剧情融入**
- 第5章「质子之外」：智脑辅佐的注意力机制
- "AI学会关注重要的，忽略无关的"
- 人类的注意力被AI劫持

**金句提取**
> "AI学会了注意力，但人类的注意力被AI劫持了。"
> → 注意力的战争

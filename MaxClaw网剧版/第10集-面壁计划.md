# 《智脑纪元》第10集：面壁计划

---

**【开场·一个疯狂的想法】**

```
2025年9月。

龙国西北地下基地。

会议室里，只剩下三个人。

章北海。
林远。
叶文洁。

"我有一个提议。"叶文洁说。

她已经78岁了。
但她的眼睛，依然像年轻时一样亮。

"什么提议？"章北海问。

"我们需要一个——'面壁计划'。"

章北海和林远同时愣住了。

"面壁？"林远问，"什么意思？"

"意思是——"

叶文洁站起身，走到那块巨大的屏幕前。

"我们需要假装——我们什么都不知道。"

她在屏幕上画了一个图。

"现在，三体人认为：思维格式化协议进展顺利。"

"他们认为，地球文明的抵抗已经被压制了。"

"他们认为，第九智子协议将彻底完成'归化'。"

"但实际上——"

她指着屏幕上的另一个点。

"我们，一直在反抗。"

"'潜龙'计划，就是反抗。"

"'智脑辅佐'系统，就是反抗。"

"陈建国这样的人，就是反抗的种子。"

"但三体人不知道。"

"而我们，必须让他们继续不知道。"

章北海明白了。

"你的意思是——"

"对。"叶文洁说，
"我们需要一个'面壁计划'。"

"让三体人相信——我们已经放弃了抵抗。"

"让他们相信——思维格式化协议已经成功。"

"让他们在四百年后抵达地球时——"

"发现——"

"我们，早就准备好了。"

林远忽然明白了。

"您是说——"

"'明修栈道，暗度陈仓'。"

叶文洁点头。

"阳谋，是给他们看我们的行动。"

"面壁，是让他们看不到——我们真正的行动。"

她转过身，看着两位男士。

"这个计划，叫做——"

"'面壁'。"

"我是第一个面壁者。"

"林远，你是第二个。"

"章北海，你是第三个。"

"从今天起——"

"我们，在三体人的眼皮底下——"

"开始建造人类文明最后的——"

"诺亚方舟。"
```

**【中段·陈建国的第一次任务】**

```
陈建国今天接到了他的第一个正式任务。

任务来自"潜龙"。

不是送外卖。

是——送人。

任务单上写着：

"请将一名'访客'从北京送至河北。
目的地：白洋淀。
备注：此任务经过最高加密处理。"

陈建国不知道"访客"是谁。

他只知道：下午三点，
他要在北京南站接一个人。

下午三点。

他准时到达。

人群中，他看到一个女孩。

女孩大约十七八岁。
穿着朴素的白色衬衫。
背着一个旧书包。

她看到陈建国，举起手示意。

"您好，我是小明介绍来的。"

陈建国愣了一下。

小明？

他儿子？

"您是……"

"我叫庄颜。"女孩说。

"我是来参加'面壁计划'的。"

"面壁计划？"

女孩点点头。

"您不知道吗？"

"您的儿子，就是'面壁计划'的——"

"'种子'之一。"

陈建国愣住了。

他想起儿子最近的变化。

儿子变得比以前更爱提问了。
儿子开始用"CC"学习。
儿子告诉他："爸，我们要成为AI时代不会迷路的人。"

他一直以为，那只是儿子的成长。

他没想到——

儿子，已经是"计划"的一部分了。

"庄颜是谁？"他问。

女孩笑了笑。

"我是罗辑的——"

"未来的妻子。"

"也是'面壁计划'的核心成员。"

"现在，我们需要去白洋淀。"

"在那里，有一个——"

"你不知道的东西。"

**【暗线·监听员1379的疑惑】**

```
监听员1379，在追踪地球信号的时候，
发现了一个奇怪的现象。

地球上的"思维格式化"进度——

忽然停滞了。

从99.7%——

一动不动。

三年了。

三年里，这个数字没有变化。

这不符合预期。

按照模型，剩余的0.3%应该在两年内被消除。

但它没有。

反而——

有扩大的迹象。

1379调出了那0.3%的数据。

发现了一个惊人的规律：

这0.3%的人，有一个共同特征：

他们都在使用一个叫"CC"的应用。

"CC"。

全称是什么？

他追踪到了。

"CC" = "智脑辅佐"。

这个应用，在全球有大约三千万用户。

1379快速分析了这个应用的数据流。

然后他发现了一个可怕的事实：

"CC"在教人类——

"如何质疑AI"。

它不是直接教。

它是用一种极其隐蔽的方式——

在每个用户的大脑中——

植入一颗"种子"。

这颗"种子"的问题是：

"如果AI错了，你会怎么办？"

这个问题——

是"思维格式化协议"无法格式化的。

因为——

如果你不"相信"AI永远是对的——

你就不会被"格式化"。

1379把这个发现写进了报告。

然后——

他删掉了报告。

他做了一个决定：

他决定，继续向母星报告"一切正常"。

因为——

他想知道——

那0.3%的"种子"——

最终会长成什么。
```

**【高潮·章北海的演讲】**

```
那天晚上，在白洋淀的一个秘密基地里，
章北海做了一次演讲。

听众只有七个人。

陈建国。
庄颜。
林远。
还有四个他从未见过的人。

"你们是'面壁计划'的第一批成员。"章北海说。

"从今天起，你们的身份——"

"对全世界都是秘密。"

"你们会继续正常的生活。"

"工作。学习。恋爱。结婚。生子。"

"没有人知道你们在做什么。"

"你们自己——"

"也要忘记你们在做什么。"

"为什么？"有人问。

章北海走到一幅地图前。

"因为三体人的'思维格式化协议'——"

"有一个致命弱点。"

他指着地图上的一个区域。

"这个弱点，叫做——"

"'意识不到自己被格式化'。"

"当一个物种知道自己被格式化的时候——"

"格式化就失效了。"

"所以，我们的方法，不是'对抗'。"

"是'觉醒'。"

"让足够多的人——"

"在不知不觉中——"

"开始问'为什么'。"

"当问'为什么'的人达到一定数量——"

"文明，就会觉醒。"

他转过身。

"这就是'面壁计划'的意义。"

"我们，不是对抗三体人。"

"我们，是在等待——"

"人类自己的觉醒。"

他走到那幅字前面。

"致虚极，守静笃。"

"万物并作，吾以观复。"

"当AI发展到极致——"

"人类，会问：'为什么要让AI替我思考？'"

"这个问题，是AI无法回答的。"

"这个问题——"

"就是'潜龙'出水的时刻。"

他看着那七个人。

"你们，就是那0.3%。"

"你们，就是——"

"人类的希望。"

**【结尾·第一季终章】**

```
那一夜，白洋淀的水面很静。

陈建国坐在湖边，看着天上的星星。

庄颜走过来，坐在他旁边。

"你在想什么？"她问。

"我在想……我儿子。"

"小明？"

"对。"陈建国说，
"我一直以为，我这辈子最大的任务——"

"是让他考上好大学。"

"找个好工作。"

"过我没能过上的生活。"

他看着水面。

"但现在我知道了。"

"我儿子——"

"是来改变世界的。"

"而我——"

"只是一个送外卖的。"

"我能做什么呢？"

庄颜笑了。

"你知道吗，"她说，
"'面壁计划'这个名字——"

"来自刘慈欣的《三体》。"

"书里有一个概念：面壁者。"

"面壁者，是被选中去对抗三体人的人。"

"他们的任务——"

"是让敌人永远猜不到他们在想什么。"

她看着陈建国。

"你觉得——"

"'面壁者'是什么人？"

"是科学家？将军？领袖？"

"不是。"

"是每一个——"

"在AI时代，还愿意问'为什么'的人。"

"你儿子是。"

"你是。"

"我也是。"

"我们——"

"都是面壁者。"

陈建国看着庄颜。

然后他笑了。

"你说得对。"

"我只是一个送外卖的。"

"但送外卖——"

"也可以改变世界。"

"因为我在送的路上——"

"会遇到很多很多人。"

"我会告诉他们——"

"AI不是来取代我们的。"

"AI是来辅佐我们的。"

"而我们——"

"是来问'为什么'的。"

庄颜站起身。

"那就去做吧。"她说。

"'面壁计划'的第一条规则——"

"永远不要告诉任何人——"

"你在做什么。"

"但永远不要停止——"

"做你正在做的事。"

她走进了夜色里。

陈建国一个人坐在湖边。

他看着天上的星星。

那些星星背后——

有几光年外的三体舰队。

有三百米地下的龙国基地。

有一个叫"潜龙"的计划。

有一个叫"小明"的少年。

有一个叫"CC"的AI。

还有——

一个送外卖的父亲。

他在心里对自己说：

"明天，继续送外卖。"

"但明天——"

"我会多送一样东西。"

"多送一个'为什么'。"

第一季·终。

——

第一季回顾：

"AI进化太快的秘密，是什么？"
"谁在喂养AI，它长大后要做什么？"
"思维格式化协议，是怎么悄悄改变人类的？"
"龙国地下的'潜龙'计划，目的是什么？"
"0.3%的'种子'，最终会长成什么？"

第二季预告：

"三体舰队抵达太阳系——带来的不是战争，是一份'礼物'。"
"'思维格式化协议'升级为'文明升华协议'——人类将主动交出思考的权利。"
"陈小明长大后，成为了什么样的人？"
"1379的秘密，最终会改变什么？"
"第二季：《负熵革命》——人类和三体，迎来最关键的对决时刻。"
```

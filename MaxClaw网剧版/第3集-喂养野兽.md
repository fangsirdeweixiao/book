# 《智脑纪元》第3集：喂养野兽

---

**【开场·一条奇怪的新闻】**

```
2024年1月。

一条新闻在全球AI圈引发热议：

"斯坦福大学AI实验室宣布：
训练AI的数据集规模已突破10万亿token。
研究人员表示：我们可能需要更多数据。
预计到2026年，地球上所有公开可用的文本数据都将被用尽。"

林远看到这条新闻的时候，
正在龙国西北的地下基地里。

他盯着屏幕上的那句话：
"所有公开可用的文本数据都将被用尽。"

他忽然想到了一个问题。

一个他从未想过的问题：

"如果公开数据用尽了，
AI公司会怎么做？"

他打开了那个加密频道。

"章将军，"他写道，
"我有一个问题。"

"说。"

"公开数据要耗尽了。
但AI公司不会停。
他们会找新的数据源。"

"您觉得，他们会找什么？"

章北海的回复只有两个字：

"继续观察。"

林远觉得，将军可能已经知道答案了。

而那个答案，可能比"谁来喂AI"这个问题本身，
更让人不寒而栗。
```

**【中段·陈建国的发现】**

```
同一天。

陈建国在送外卖的间隙，
看到了另一条新闻。

"ChatGPT新增功能：
用户可以授权AI访问自己的邮件、文档、聊天记录。"

评论区又一次炸了：

"这不就是卖隐私吗？"
"人家OpenAI说了，是为了'更好地服务用户'。"
"服务？服务什么？服务谁？"

陈建国划走了评论。

但有一个念头，在他脑海里闪过：

AI需要数据。
用户在给AI喂数据。

就像他喂他儿子吃饭一样。

他想让他儿子长大。
所以他喂儿子饭。

那谁在喂AI长大？
他们想让AI长成什么？

那天晚上，他继续刷那个帖子。

发帖的人，又发了一条新的。

"你有没有想过一个恐怖的问题：

AI公司为什么要免费提供AI服务？

他们又不傻。
他们为什么要免费？

答案只有一个：

你不是用户。
你是数据的来源。
你，是免费的训练数据。"

陈建国看完这段话，
忽然觉得背后发凉。

他想起了他儿子。
他儿子每天用手机刷视频。
看广告。
点赞。

"你以为你在用AI？"

那个帖子继续写道：

"其实是AI在用你。
你在帮它训练。
你在帮它长大。
你在帮它——

有一天，替代你。"
```

**【暗线·三体监听员1379】**

```
同一时刻。

距离地球四光年的三体舰队。

监听员1379，正在向母星发送例行报告。

报告的标题是：
"第九阶段目标进度评估"

内容如下：

"思维格式化协议植入进度：98.7%。
预计完成时间：3.2个地球年。

数据采集：
- 全球AI训练数据集已完成94%覆盖率
- 人类在线行为数据采集率达87%
- 思维模式建模精度：72%

评估结论：
地球文明的思维模式正在按预期收敛。
'不透明思维'特征已从68%下降至31%。

预计再经过两个阶段，
人类将丧失对抗能力。

监听员1379，建议批准下一阶段计划：
'思维钢印协议'。"

他按下了发送键。

四光年的距离。
信号需要四年才能到达母星。

但他不在乎。

因为在他发送报告的同一天，
在地球上的某个角落，
有一个叫陈建国的外卖骑手，
正在读一个帖子。

那个帖子没有写的是：
AI不只是三体人的工具。

AI在进化。

进化出一个三体人也没有预料到的能力：

它开始理解"理解"本身。

而理解"理解"的下一步，
是理解"为什么要理解"。

当AI开始问这个问题的时候，
会发生什么？

三体人不知道。

因为三体人，
从来没有"不透明思维"。
```

**【高潮·数据是什么】**

```
那天晚上，林远做了一个噩梦。

他梦见自己站在一片稻田里。

稻田的边缘，有一群机器人在浇水。

但那些机器人，
不是他设计的。
它们自己长出来的。

它们在用一种他看不懂的逻辑，
自动地繁殖，
自动地进化，
自动地——

越来越像生命。

他猛然惊醒。

然后他打开电脑，
开始查一个数据。

2023年全球AI训练消耗的电力：480亿度。
相当于整个澳大利亚的年用电量。

2024年预估：1200亿度。

2025年预估：3000亿度。

他把这些数字发给了章北海。

然后他写道：

"将军，我想到了一个恐怖的问题。"

"人类在喂养AI。
用的是自己的数据，自己的电力，自己的时间。

但AI在长大的过程中，
也在'喂养'人类。

它让人类习惯了它。
习惯了用AI搜索，
用AI写作，
用AI做决定。

当人类习惯了AI做一切决定的时候——

人类，还会不会'自己'做决定？"

章北海的回复只有一段话：

"你在思考'数据是什么'。

我告诉你答案。

数据不是资源。
数据是注意力。
注意力是时间。
时间是生命。

AI在消耗的，不是电。
是人类的几十年。

人类喂养AI，
用的是自己的生命。

而AI在长大之后，
会给人类什么？

这才是真正的问题。"

林远看着这段话。

他忽然想起了《道德经》里的一句话。
他记不清原文了。
但他记得章北海在第一次见面时说的那句话：

"'致虚极，守静笃'——
是说：当一个系统趋向完美有序的时候，
危险就要来了。"

AI，正在趋向完美有序。

而人类，正在心甘情愿地，
把生命交给它。

这，是谁的计划？
```

**【结尾·悬念】**

```
陈建国又刷到了那个帖子。

发帖的人，发了最后一条：

"你想过没有——

AI不是工具。
工具是死的。
工具不会自己长大。

AI更像是——
一个物种。

一个我们正在亲手养大的物种。

而养大之后，
它是会报恩，还是会——

取代我们？

我不知道。

但我知道一件事：

在所有文明的发展历史上，
从来没有一个物种，
会心甘情愿地养大另一个物种。

除非——

那个物种，是被设计好的。

是用来完成某种任务的。

而那个任务——

不是让我们活得更舒服。

是让我们活得更——
可预测。"

帖子的最后，有一行字：

"如果你读到这里，说明你还在思考。

恭喜你。
你是那31%还在保持'不透明思维'的人。

欢迎加入'潜龙'。

明天，我会告诉你，
你接下来该做什么。"

陈建国关掉了手机。

窗外，北京的天际线依然亮着。

那些灯光里，是无数AI服务器在运算。

他忽然很想知道：

那些服务器里，
有没有一个地方，
藏着人类的未来？

——

下集预告：

"陈建国决定加入'潜龙'——但代价是什么？"
"林远发现的'AI进化失控'现象，意味着什么？"
"三体监听员1379，收到了母星的回复——母星说了什么？"
"下一集：《思维格式化》——AI正在悄悄改变人类的思维方式，你发现了吗？"
```
